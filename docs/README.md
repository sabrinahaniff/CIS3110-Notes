# CIS*3110: Operating Systems

Free, open-source study notes for CIS*3110 at the University of Guelph. Covers all major course topics with diagrams, worked examples, and quick reference tables.

> Notes by **Sabrina Haniff**: feel free to use, share, and contribute.

---

## What's Covered

| Topic | Description |
|-------|-------------|
| [Introduction to OS](01-introduction.md) | OS role, system calls, user/kernel mode, process states |
| [Processes](02-processes.md) | PCB, process states, fork/exec/wait, IPC, pipes |
| [CPU Scheduling](03-cpu-scheduling.md) | FCFS, SJF, SRTF, Round Robin, Priority scheduling |
| [Synchronization](04-synchronization.md) | Critical sections, semaphores, mutexes, classic problems |
| [Deadlock](05-deadlock.md) | Coffman conditions, resource allocation graphs, Banker's algorithm |
| [Memory Management](06-memory-management.md) | Paging, TLB, segmentation, address translation |
| [Virtual Memory](07-virtual-memory.md) | Demand paging, page faults, page replacement algorithms |
| [File Systems](08-file-systems.md) | File allocation, inodes, directories, permissions |
